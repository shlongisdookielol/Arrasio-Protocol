# Arrasio Protocol (shlong#2873 and Cesnad#0803)
### Outgoing Packets
| Header  | Information       |
|---------|-------------------|
| `C`     | Camera Packet     |
| `d`     | Ticks Camera      |
| `p`     | Ping              |
| `x`     | Upgrade Stats     |
| `U`     | Upgrade Tank      |
| `s`     | Spawn With Name   |
| `L`     | Leveling Up       |

### Incoming Packets
| Header  | Information       |
|---------|-------------------|
| `u`     | update            |
| `b`     | Broadcast         |
