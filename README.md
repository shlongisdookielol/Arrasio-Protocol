# Arrasio Protocol
### Outgoing Packets
| Header  | Information       |
|---------|-------------------|
| `C`     | Camera Packet     |
| `d`     | Ticks Camera      |
| `p`     | Ping              |
| `x`     | Upgrade Stats     |
| `U`     | Upgrade Tank      |
| `s`     | Spawn With Name   |
| `L`     | Leveling Up       |

### Outgoing Packets
| Header  | Information       |
|---------|-------------------|
| `u`     | Maybe Position    |
| `b`     | Leader (I Think)  |
